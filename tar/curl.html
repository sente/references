curl

grab_url

grab_url can be used to set the cookie for a user, like this: 

function grab_url  {
	URL=$1
	OUT=$2
	echo $URL
	echo $OUT
	if [ -z "$OUT" ]; then 
		echo "curl -s $URL"
		curl -s $URL 
	else
		echo "curl -s $URL -o $OUT"
		curl -s $URL -o $OUT
	fi
}

Calling the above class with GET will set a cookie named "age" with a default value equal to "25" (this default value actually comes from the web.input processing, NOT the setcookie function), which expires in 1 hour (3600 seconds).

upload_two_files

upload_two_files can be used to set the cookie for a user, like this: 

function upload_two_files() {
	TARGET=http://sente.cc/stu/cgi-bin/compare-bin.py
	FILE=/etc/passwd
	FILE2=/etc/group
	curl -F "file.1=@${FILE}" "file.2=@${FILE}" $TARGET
}

Calling the above class with GET will set a cookie named "age" with a default value equal to "25" (this default value actually comes from the web.input processing, NOT the setcookie function), which expires in 1 hour (3600 seconds).
